# Air-Quality-Index-Prediction
Machine Learning project to predict Air Quality Index (AQI) using environmental pollutant data. Built with Python, Scikit-learn, Flask API, and SQLite.
# Air Quality Index Prediction

## Overview

This project predicts Air Quality Index (AQI) using Machine Learning.

## Dataset

50,000 environmental records

## Technologies

Python
Scikit-learn
Pandas
NumPy
Flask
SQLite

## ML Model

Random Forest Regressor

## Evaluation

R² Score : 0.81
MAE : 1.89
MSE : 27.54

## Features

CO
CO2
NO2
SO2
O3
PM2.5
PM10
Hour
Day
Month
Weekday

## Run

python train.py

python app.py
